Contains the value of pi.

Synopsis

$ZPI  

Description

$ZPI contains the value of the numeric constant Pi to eighteen decimal places (3.141592653589793238).

This value is frequently used with trigonometric functions, such as the sine function [$ZSIN](http://localhost:57772/csp/docbook/DocBook.UI.Page.cls?DocBook.UI.Page.cls?KEY=RCOS_fzsin).
